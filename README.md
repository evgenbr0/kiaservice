# KIA Service — сайт автосервиса (kiaservice.ru)

Готовый статический сайт дружелюбного автосервиса по обслуживанию и ремонту автомобилей
KIA — «сервис у дома». Сделан по техническому заданию: чистый HTML/CSS/JS, без бэкенда,
базы данных и сборщиков. Полностью адаптивный, работает сразу после копирования на хостинг.

---

## 📁 Структура проекта

```
kiaservice.ru/
├── index.html              # Главная (и единственная) страница сайта
├── .nojekyll               # Отключает обработку Jekyll на GitHub Pages
├── assets/
│   ├── css/
│   │   └── style.css       # Все стили
│   ├── js/
│   │   └── main.js         # Интерактив: меню, форма, FAQ, фильтр цен, анимации
│   └── img/
│       └── favicon.svg     # Иконка сайта (округлый коралловый знак KIA)
└── README.md               # Этот файл
```

---

## 🚀 Как разместить на хостинге (инструкция)

Скопируйте всю папку `kiaservice.ru` (файл `index.html`, файл `.nojekyll` и папку `assets`)
в корневую директорию вашего хостинга — обычно это `public_html`, `www` или `htdocs`. Файл
`index.html` должен лежать в корне, а папка `assets` — рядом с ним. После загрузки сайт сразу
откроется по адресу домена без какой-либо настройки сервера, сборки или установки зависимостей.
Никаких баз данных и серверных языков не требуется.

> Локально сайт также можно открыть, просто дважды кликнув по `index.html` в браузере.

## ✨ Что есть на сайте

- **Главный экран** с тёплым позиционированием («сервис KIA рядом с домом») и карточкой онлайн-записи с пастельными бейджами.
- **Услуги** — 8 услуг в виде крупных скруглённых карточек с цветными «чип»-иконками в кружках.
- **Почему мы** — «пузырьковые» (bubble) карточки преимуществ с мягкими тёплыми тенями.
- **Модели KIA**, которые обслуживаются: Rio, Sportage, Ceed, Sorento, Soul, Picanto, Seltos, Cerato, K5, Optima.
- **Цены** — понятный прайс с фильтром-пилюлями по категориям (ТО, двигатель, подвеска, тормоза, электрика, шины).
- **Как это работает** — скруглённый таймлайн из 6 шагов с крупными номерами в кружках.
- **Отзывы** клиентов с круглыми цветными аватарами.
- **Форма записи** с валидацией, маской телефона и сообщением об успешной отправке.
- **FAQ** — мягкий аккордеон с частыми вопросами.
- **Контакты** с тёплыми карточками и схемой проезда (плейсхолдер).
- **Адаптив**, мобильное меню-бургер, плавающая панель действий (Позвонить / Записаться), кнопка «наверх».

Все контактные данные (телефон, адрес, e-mail, мессенджеры, реквизиты) намеренно
заменены заглушками **ХХХХХ** согласно ТЗ — их нужно заменить на реальные перед публикацией.

---

## 🔧 Что заменить перед запуском

1. Контактные заглушки `ХХХХХ` → реальные телефон, адрес, e-mail, мессенджеры, реквизиты
   (в `index.html`, включая блок Schema.org в `<head>` и значения `tel:`/`mailto:`).
2. При необходимости — реальную карту (Яндекс.Карты / Google Maps) вместо плейсхолдера.
3. Цены в прайсе — на актуальные (сейчас демонстрационные).
4. Форма записи имитирует отправку на стороне браузера. Чтобы заявки приходили на почту/в
   CRM, подключите сторонний обработчик форм (например, Formspree).

## 🛠 Технологии

- Семантический **HTML5**, разметка **Schema.org** (AutoRepair) для SEO.
- **CSS3**: кастомные свойства, Grid/Flexbox, большие радиусы и мягкие тени, «pill»-эстетика,
  адаптивные сетки, анимации, `prefers-reduced-motion`.
- **Ванильный JavaScript** (без библиотек): фильтр прайса по категориям, валидация форм
  с доступностью (aria-invalid, role=alert), маска телефона, мобильное меню с управлением
  фокусом (focus-trap, inert), аккордеон FAQ, анимации по скроллу, кнопка «наверх».
- Шрифт **Nunito** (с кириллицей) подключён через Google Fonts (CDN). Есть системные
  фолбэки — сайт остаётся читаемым даже без доступа к CDN.

Внешних зависимостей, требующих сборки или установки, **нет**.
